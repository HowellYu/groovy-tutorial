# [%Title]

[%Abstract] 

The following libraries are imported into Groovy by default:

- `java.io.*`
- `java.lang.*`
- `java.math.BigDecimal`
- `java.math.BigInteger`
- `java.net.*`
- `java.util.*`
- `groovy.lang.*`
- `groovy.util.*`

Scripts are able to make use of classes within these packages without explicitly importing them.
